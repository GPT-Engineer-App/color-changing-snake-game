# color-changing-snake-game

create a browser-based snake game. change the color of the background each time the snake eats something. use a smily emoji for the food. show a message with "Awesome!" each time the snake eats something.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/color-changing-snake-game.git
cd color-changing-snake-game
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
